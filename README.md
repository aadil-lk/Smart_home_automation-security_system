Smart Home Automation and Security System
Project Description

The Smart Home Automation and Security System is an Arduino Uno based IoT project designed to provide users with intelligent home control and enhanced security features through a smartphone application. The system enables wireless communication using Bluetooth and Wi-Fi technology, allowing users to remotely monitor and control household devices such as lights and fans directly from their mobile device.

In addition to automation, the project integrates safety and security functionalities using multiple sensors. A gas sensor is used to detect hazardous gas leaks, while a proximity sensor is implemented to identify potential intrusions or unauthorized movements. Whenever a gas leak or intrusion event is detected, the system instantly sends alerts and notifications to the mobile application, allowing users to respond quickly and improve home safety.

The project combines automation, real-time monitoring, and security into a single smart solution. The use of Arduino Uno along with wireless communication modules and sensor-based detection demonstrates the practical implementation of Internet of Things (IoT) concepts for creating a connected and intelligent home environment.

Key Features
Remote control of home appliances such as lights and fans
Bluetooth and Wi-Fi based communication
Mobile application for monitoring and device control
Real-time gas leak detection
Intruder detection using proximity sensors
Instant mobile notifications and alerts
Arduino Uno based implementation
Smart and user-friendly home security solution
